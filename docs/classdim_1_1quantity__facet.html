Detailed Description

template<class Scalar, class System>
class dim::quantity_facet< Scalar, System >

An iostream facet for formatting quantities. This is typically used by operator<< and operator>>, but can be adjusted to change how quantities are formated in a program.

Note
This is not a functional facet, but rather a base class for such facets. To make a functional facet:
class MyFacet : public quantity_facet<MyScalar, MySystem> {
public:
static std::locale::id id;
};
